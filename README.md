# ExBitfinex

Bitfinex API Client for Elixir

## Installation

Add the `ex_bitfinex` package to your list of dependencies in `mix.exs`:

```elixir
def deps do
  [{:ex_bitfinex, "~> 0.0.1"}]
end
```
